पहिलो AI Project!
Sentiment Analysis — खुशी कि दुखी?
🎯 लक्ष्य

हामी एउटा AI बनाउँछौं जसले English वाक्य पढेर भन्न सक्छ — "यो वाक्य खुशीको (POSITIVE) हो कि दुखीको (NEGATIVE)।" यसलाई Sentiment Analysis भनिन्छ!

नयाँ cell मा यो पूरै code राख्नुहोस्:

from transformers import pipeline

# AI model load गर्ने
classifier = pipeline("sentiment-analysis")

# वाक्यहरू जाँच गर्ने
sentences = [
    "I am very happy today!",
    "This is so sad and terrible.",
    "I love learning AI!"
]

for sentence in sentences:
    result = classifier(sentence)
    print(f"वाक्य: {sentence}")
    print(f"भावना: {result[0]['label']} ({result[0]['score']:.0%})")
    print("---")

✓ output यस्तो आउनु पर्छ:

Output
वाक्य: I am very happy today!
भावना: POSITIVE (100%)
---
वाक्य: This is so sad and terrible.
भावना: NEGATIVE (100%)
---
वाक्य: I love learning AI!
भावना: POSITIVE (100%)
---

Code को अर्थ:

Codeअर्थ
pipeline(...)AI model तयार गर्छ
classifier(...)वाक्य AI लाई दिन्छ र जवाफ लिन्छ
for sentenceएक-एक वाक्य जाँच गर्छ (loop)
result[0]['label']POSITIVE वा NEGATIVE
result[0]['score']AI कति निश्चित छ — percentage
